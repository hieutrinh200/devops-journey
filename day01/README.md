# Day 01 - Linux basics & Bandit level 0

## Key terms (EN/VI)

- Terminal: A command-line interface window to interact with the OS using text commands.  
  → Terminal: Cửa sổ dòng lệnh để tương tác với hệ điều hành bằng lệnh chữ.

- Shell: Program that interprets commands typed in the terminal (e.g. bash).  
  → Shell: Chương trình nhận và thực thi lệnh trong terminal (ví dụ: bash).

- Command: A single instruction given to the shell (e.g. `ls`, `cd`).  
  → Command (lệnh): Một câu lệnh gửi cho shell (ví dụ: `ls`, `cd`).

- Filesystem: The way files/folders are organized on disk (/, /home, /tmp, …).  
  → Filesystem (hệ thống tệp): Cách tổ chức file/thư mục trên ổ đĩa (/, /home, /tmp,…).

- Path: The address of a file or directory (e.g. `/home/user/devops-day1`).  
  → Path (đường dẫn): Địa chỉ của file/thư mục (ví dụ: `/home/user/devops-day1`).

- Home directory: Default personal directory of a user (e.g. `/home/username`).  
  → Home directory (thư mục home): Thư mục cá nhân mặc định của người dùng.

- SSH (Secure Shell): Encrypted protocol to remotely access a server.  
  → SSH: Giao thức mã hóa để truy cập điều khiển máy chủ từ xa.
