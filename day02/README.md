# Day 02 - Paths, execute permission & first Bash script

## Key terms (EN/VI)

- Absolute path: A full path starting from the root `/`, e.g. `/home/user/devops-journey/day02`.
  → Đường dẫn tuyệt đối: Đường dẫn đầy đủ bắt đầu từ thư mục gốc `/`, ví dụ `/home/user/devops-journey/day02`.

- Relative path: A path relative to the current working directory, e.g. `../day02`.
  → Đường dẫn tương đối: Đường dẫn tính từ thư mục hiện tại, ví dụ `../day02`.

- Current working directory: The directory where your shell is currently “standing”.
  → Thư mục làm việc hiện tại: Thư mục mà shell “đang đứng” tại đó.

- Execute permission (`x`): Permission that allows a file to be executed as a program/script.
  → Quyền thực thi (`x`): Quyền cho phép một file được chạy như chương trình/script.

- Shebang (`#!/usr/bin/env bash`): Special line that tells the system which interpreter to use.
  → Shebang: Dòng đầu file chỉ cho hệ thống biết dùng trình thông dịch nào để chạy script.

- Script: A text file containing commands to be executed in sequence.
  → Script: File văn bản chứa nhiều lệnh sẽ được thực thi lần lượt.
